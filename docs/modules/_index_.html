External module "index"

This is the main module of timecount and contains objects that can manipulate time values, convert them, and write them in many different formats.

The interface NanosecondBasedTime is the base unit of time of timecount. All other units are calculated from nanoseconds.

See the other modules of timecount: localization and utils for more information.

TimeUnitParameter

TimeUnitParameter: keyof TimeUnitDatabase | TimeUnit
  • Defined in index.ts:817

A union type, used by parameters that accept either a time unit name or a TimeUnit object.

The accepted names are all the properties defined by TimeUnitDatabase.

Const timeUnits

timeUnits: object
  • Defined in index.ts:77

A database of TimeUnit objects, used to measure and represent time values in different formats. Each property is an object describing the time unit's nanosecond correlation (TimeUnit.factor), its linguistic properties for English (US) and its symbol, if it exists.

See TimeUnitDatabase for more information about time units.
